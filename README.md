# WanderHive# WanderHive 🐝

WanderHive is a full-stack travel listing web application where users can explore, create, and review travel destinations.


## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript, EJS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB, Mongoose
- **Authentication:** Passport.js
- **Maps API:** Mapbox
- **Templating Engine:** EJS + EJS-Mate
- **Deployment:** Render

## 📸 Features

- 🗺️ View travel listings with location map
- ✍️ Register/Login functionality
- 🧳 Add new listings
- ⭐ Post reviews and ratings
- 🔐 Authentication & session handling
- 🗑️ Delete your listings and reviews
- 💾 Flash messages for user feedback
